# VC2Starter

A lightweight configuration and patch utility for **Vietcong 2** that enables several fixes and quality-of-life improvements unavailable in the original game.

## Features

* ✅ Enable Hardware Rendering
* ✅ Fix "CD key already used" error
* ✅ Skip Intro Videos
* ✅ FPS Limiter (30–120 FPS)
* ✅ Show FPS Counter on Game Start

## About

VC2Starter Configurator is a small utility designed to improve the Vietcong 2 experience by enabling hidden features and applying fixes to common issues.

### Hardware Rendering

Enables hardware rendering support, improving compatibility and performance on modern systems.

### CD Key Fix

Fixes the annoying **"CD key already used"** error that may appear even when using a valid game key.

### Skip Intro

Skips the game's startup videos, allowing you to get into the game faster.

### FPS Limiter

Lets you limit the game's framerate between **30 and 120 FPS**.

### FPS Counter

Displays the current FPS directly after launching the game.

## Usage

1. Launch `VC2StarterConfigurator`.
2. Select the desired options.
3. Click **OK**.
4. Start the game.
